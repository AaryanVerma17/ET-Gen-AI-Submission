# 🚀 QUICK START GUIDE

1. **Install requirements:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Run the dashboard:**
   ```bash
   streamlit run dashboard/app.py
   ```

3. **Use the App:**
   - Enter your financial details in the form.
   - Instantly see your Money Health Score, category breakdown, insights, and recommendations.

---

## 📝 Project Structure
- app.py – Streamlit UI
- score_calculator.py – Scoring logic
- insight_engine.py – Insight generation
- recommendation_engine.py – Recommendations
- money_health.py – (Optional) Legacy/compatibility logic

---

## 📝 Requirements
- Python 3.10+
- streamlit
- plotly
